======How to Run====
1. Open CMD.
2. Type in >java -jar [Path to file]\PencilDurability.jar
3. Follow the Steps listen in the program
======FEATURES======
1. Write allows you to write at the end of what is already written(including spaces).
2. Erase allows you to replace the last occurence of a string or character with spaces.
3. Sharpen resets point durability but also reduces the pencil length by 1.
4. Create New Pencil allows you to create a new pencil if you don't want to use your current one anymore.
5. Edit last gap allows you to fill the last gap in the text to new text printing out '@' anywhere that already contains a character.
6. Exit closes the application.
